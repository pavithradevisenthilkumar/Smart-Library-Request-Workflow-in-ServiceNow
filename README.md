# 📚 Smart Library Request Workflow – ServiceNow (Short)

## 📌 Overview

A ServiceNow workflow to automate library requests like book issue, return, renewal, and new book requests.

## 🔄 Process

1. User submits request via Service Catalog
2. System validates request (availability & eligibility)
3. Sent to librarian for approval
4. Task assigned to library staff
5. Request processed (issue/return/renew)
6. Status updated and user notified

## ⚙️ Key Components

* Service Catalog (Library Request)
* Custom Table (`u_library_request`)
* Flow Designer
* Notifications

## 👥 Roles

* Admin – Full access
* Librarian – Approves requests
* User – Submits requests

## ✅ Benefits

* Saves time
* Reduces manual work
* Easy tracking of requests

## 📝 Conclusion

This workflow simplifies and automates library management efficiently.
